#################################################################################################
!!!IMPORTANT!!!
- In order to run these scripts, this folder MUST be place inside 'test-network' folder.
- You MUST set execute permission on for all scripts in this folder:
			'chmod 755 *.sh'
#################################################################################################
- There is 1 main script to interact with: hamburg.sh (Your working directory MUST be this folder 
in order for this script to run correctly)

- The script 'orgEnv.sh' is for changing environment variables to match with a given organization
(mainly for chaincode testing purpose, this script can be run in any working directory)

For more information on how to use these 2 scripts, run these scripts with a -h flag
